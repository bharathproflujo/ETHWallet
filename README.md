# ETHWallet
Simple Wallet design to view and send ETH 

## Components
- ethers.js
- Infura or Alchemy

## Instructions

`INFURA_API_URL` - We should define the Node API's url here
`WALLET_ADDRESS` - Add your wallet address 
`WALLET_PRIVATE_KEY` - Add your wallet's private key

to run locally 

`npm start web`
